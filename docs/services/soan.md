# SOAN（草庵）

**Web フロントエンドの成果物を、ドラッグ&ドロップだけで一瞬で公開できる一時ホスティングサービスです。**

作ったページを誰かに見せたい。でもサーバーを用意するほどでもない——SOAN は、静的ファイルをドロップするだけで共有 URL を発行し、時間が来ればすべて自動で消える「仮初めの宿」を提供します。名前の「草庵」は、必要なときだけ立ち寄れる簡素な庵をイメージしています。

## こんな方におすすめ

- **Web 制作・フロントエンド開発者** — コーディング結果をクライアントや同僚にすぐ見せたい
- **デザイナー** — HTML/CSS で組んだモックアップをリンク一つで共有したい
- **学習中のエンジニア** — 課題やポートフォリオを一時的に公開してフィードバックをもらいたい
- **チーム開発** — PR レビュー前にビルド成果物をサッと確認してもらいたい

## 主な機能

### ドラッグ&ドロップで即公開

ファイルまたはディレクトリをドロップエリアに入れるだけ。HTML / CSS / JS / 画像・フォントなど主要なフロントエンド資材に対応しています。アカウント登録なしでもすぐに使えます。

![SOANのトップ画面。ファイルのドロップエリアとディレクトリ選択ボタン、GitHubとGoogleのログインボタンが表示されている](/images/services/soan-top.png)

### ファイル選択から公開まで数クリック

ファイルを選んだら、利用規約に同意して「公開する」を押すだけ。ファイル数やサイズもその場で確認できます。ゲストは 6 時間固定、ログインすれば最大 5 日間まで公開期間を選べます。

![ファイル選択後の画面。ファイル数・合計サイズの表示、公開期間の案内、利用規約への同意チェックボックス、公開するボタンが並んでいる](/images/services/soan-upload.png)

### 共有 URL & QR コードを即時発行

公開が完了すると専用の URL が発行されます。デスクトップでは QR コードも表示されるので、スマートフォンでの確認も簡単。URL をコピーして相手に送るだけで共有完了です。

![公開完了画面。発行されたURL、残り時間のカウントダウン、QRコードが表示されている](/images/services/soan-deployed.png)

### 自動クリーンアップ

公開期間が終了すると、アップロードされたファイルは自動的に削除されます。手動で消し忘れる心配も、不要なデータが残り続けることもありません。

### その他の特徴

- **ダーク / ライトテーマ** — システム設定に連動した自動切替に対応
- **日本語 / 英語** — ブラウザの言語設定を自動検出して切り替え
- **レスポンシブデザイン** — スマートフォンからでも利用可能

![ダークモードで表示されたSOANのトップ画面](/images/services/soan-dark.png)

## 利用の流れ

1. **ファイルを選択** — ドロップエリアにファイルをドラッグ、またはクリックして選択します
2. **内容を確認** — ファイル数・合計サイズが表示されます（1 サイトあたり最大 100 ファイル / 50 MB）
3. **公開期間を設定**（ログインユーザーのみ） — 1 時間〜5 日間の範囲で選択できます
4. **利用規約に同意して公開** — 「公開する」ボタンを押すと即座に URL が発行されます
5. **URL を共有** — リンクをコピーして相手に送ります。QR コードも利用可能です

## ゲストとログインユーザーの違い

| | ゲスト | ログインユーザー |
|---|---|---|
| **アカウント登録** | 不要 | GitHub または Google で認証 |
| **公開期間** | 6 時間（固定） | 最大 5 日間（1 時間単位で選択可） |
| **マイサイト管理** | — | 作成したサイトの一覧・手動削除が可能 |
| **アップロード制限** | 最大 100 ファイル / 50 MB | 同一 |

## 対応ファイル形式

HTML, CSS, JavaScript, JSON, PNG, JPG, GIF, SVG, WebP, ICO, WOFF, WOFF2, TTF, EOT, OTF, TXT, XML, Web Manifest など、フロントエンド開発で使用する主要な形式に対応しています。
