# SOAN（草庵）

**Web 制作物・Markdown ドキュメント・ファイル一式を、ドラッグ&ドロップだけで一瞬で公開できる一時ホスティングサービスです。**

作ったページを誰かに見せたい、設計書を URL で渡したい、素材一式を共有したい——SOAN は、ファイルをドロップするだけで共有 URL を発行し、時間が来ればすべて自動で消える「仮初めの宿」を提供します。パスワード保護にも対応しているので、限定的に共有したいケースでも安心です。名前の「草庵」は、必要なときだけ立ち寄れる簡素な庵をイメージしています。

## こんな方におすすめ

- **Web 制作・フロントエンド開発者** — コーディング結果をクライアントや同僚にすぐ見せたい
- **エンジニア** — 設計書・議事録・README を URL で素早く共有したい
- **デザイナー / 受託** — 試作の見た目をクライアントに即共有したい（パスワード保護も可能）
- **イベント運営** — 配布資料を QR コードで参加者に渡したい
- **教育・講師** — 講義資料（Markdown + 数式 + 図）を生徒に短期間共有したい
- **学習中のエンジニア** — 課題やポートフォリオを一時的に公開してフィードバックをもらいたい
- **チーム開発** — PR レビュー前にビルド成果物をサッと確認してもらいたい

## 主な機能

### ドラッグ&ドロップで即公開

ファイルまたはディレクトリをドロップエリアに入れるだけ。HTML / CSS / JS / 画像・フォント、そして Markdown など主要なフロントエンド資材・ドキュメント形式に対応しています。アカウント登録なしでもすぐに使えます。

![SOANのトップ画面。ファイルのドロップエリアとディレクトリ選択ボタン、GitHubとGoogleのログインボタンが表示されている](/images/services/soan-top.png)

### ファイル選択から公開まで数クリック

ファイルを選んだら、公開期間とパスワード（任意）を設定し、利用規約に同意して「公開する」を押すだけ。ファイル数やサイズもその場で確認できます。ゲストは 6 時間固定、ログインすれば 1 時間〜5 日間の範囲で公開期間を選択でき、サイトパスワードも設定できます。

![ファイル選択後の画面。ファイル数・合計サイズの表示、公開期間の選択ボタン、サイトパスワード入力欄、利用規約への同意チェックボックス、公開するボタンが並んでいる](/images/services/soan-upload.png)

### 共有 URL & QR コードを即時発行

公開が完了すると専用の URL が発行されます。デスクトップでは QR コードも表示されるので、スマートフォンでの確認も簡単。URL をコピーして相手に送るだけで共有完了です。

![公開完了画面。発行されたURL、残り時間のカウントダウン、QRコードが表示されている](/images/services/soan-deployed.png)

### Markdown ファイルを整形ビュワーで表示

`.md` ファイルをアップロードすると、ブラウザで開いた瞬間に GitHub 風の整形されたページとして表示されます。エンジニアの設計書共有、README ドキュメント共有、議事録の素早い共有に便利です。

- GFM（テーブル / チェックボックス / 打ち消し線）対応
- シンタックスハイライト（主要言語自動検出）
- Mermaid 図描画（flowchart / sequence / state など）
- KaTeX で数式描画（インライン `$x^2$` / ブロック `$$\sum x_i$$`）
- ダーク / ライト対応
- URL 末尾に `?raw=1` を付ければ生 Markdown も取得可能

![Markdown ビュワー画面。見出し、テーブル、コードハイライト、Mermaid 図、KaTeX 数式が整形表示されている](/images/services/soan-markdown-viewer.png)

### サイトにパスワード保護をかけて共有

ログインユーザーは、アップロード時に任意のサイトパスワードを設定できます。URL を知っている人のうち、パスワードを伝えた相手だけが閲覧可能。取引先プレゼン、社内限定の試作デモなど、URL を共有してしまうと困るケースで活用できます。

- 8 文字以上の任意パスワード
- 公開期間中は Cookie で再アクセス自動許可
- 公開期間終了でパスワードごとサイト消滅
- HttpOnly + Secure Cookie + 暗号署名で保護

![パスワード保護されたサイトを訪問したときの画面。「このサイトはパスワード保護されています」の見出しとパスワード入力欄、閲覧するボタンが表示されている](/images/services/soan-password-gate.png)

### `index.html` 不要、ファイル一覧を自動生成

`index.html` を用意しなくても、SOAN がアップロードしたディレクトリ構造から自動でファイル一覧ページを生成します。クライアントへの素材渡し、ドキュメントのフォルダ共有、画像セットの確認用 URL など、「とにかくファイルが見えればいい」ケースで便利です。

- 各サブディレクトリにも一覧を自動生成
- 日本語ファイル名に対応
- ダーク / ライト対応
- 自前で `index.html` を入れた場合はそちらを優先

![自動生成されたインデックスページ。テーブル形式でファイル名とサイズが一覧表示されている](/images/services/soan-auto-index.png)

### 自動クリーンアップ

公開期間が終了すると、アップロードされたファイルは自動的に削除されます。手動で消し忘れる心配も、不要なデータが残り続けることもありません。

### その他の特徴

- **ダーク / ライトテーマ** — システム設定に連動した自動切替に対応
- **日本語 / 英語** — ブラウザの言語設定を自動検出して切り替え
- **レスポンシブデザイン** — スマートフォンからでも利用可能

![ダークモードで表示されたSOANのトップ画面](/images/services/soan-dark.png)

## 利用の流れ

1. **ファイルを選択** — ドロップエリアにファイルをドラッグ、またはクリックして選択します
2. **内容を確認** — ファイル数・合計サイズが表示されます（1 サイトあたり最大 100 ファイル / 50 MB）
3. **公開期間を設定**（ログインユーザーのみ） — 1 時間〜5 日間の範囲で選択できます
4. **パスワード設定**（任意・ログインユーザーのみ） — 8 文字以上の任意パスワードを設定できます
5. **利用規約に同意して公開** — 「公開する」ボタンを押すと即座に URL が発行されます
6. **URL を共有** — リンクをコピーして相手に送ります。QR コードも利用可能です

## ゲストとログインユーザーの違い

| | ゲスト | ログインユーザー |
|---|---|---|
| **アカウント登録** | 不要 | GitHub または Google で認証 |
| **公開期間** | 6 時間（固定） | 1 時間〜5 日間（選択可） |
| **サイトパスワード保護** | — | ◯（アップロード時に設定可能） |
| **マイサイト管理** | — | 作成したサイトの一覧・手動削除が可能 |
| **アップロード制限** | 最大 100 ファイル / 50 MB | 同一 |
| **Markdown ビュワー** | ◯ | ◯ |
| **自動 `index.html` 生成** | ◯ | ◯ |

## 対応ファイル形式

HTML, CSS, JavaScript, JSON, **Markdown (`.md`)**, PNG, JPG, GIF, SVG, WebP, ICO, WOFF, WOFF2, TTF, EOT, OTF, TXT, XML, Web Manifest など、フロントエンド開発・ドキュメント共有で使用する主要な形式に対応しています。

## よくある質問

**Q. パスワード保護したサイトを、後からパスワード変更できますか？**

A. パスワードはアップロード時のみ設定可能で、変更はできません。公開期間が短いサービスのため、変更したい場合は再アップロードをご利用ください。

**Q. Markdown ファイル内に `![](./image.png)` のような画像参照はできますか？**

A. はい、同じアップロードに画像も含めていれば動作します。Markdown と参照する画像をまとめてアップロードしてください。

**Q. 自動生成された `index.html` の表示が気に入らない場合は？**

A. ご自身で `index.html` をアップロードに含めていただければ、自動生成のものに代わってそちらが表示されます。

**Q. 公開期間が終了したサイトの URL を開くとどうなりますか？**

A. 404 ページが表示されます。ファイル本体・キャッシュとも同時に失効するため、期間後にデータが残ることはありません。
