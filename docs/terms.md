# 利用規約

本利用規約（以下、「本規約」）は、合同会社リットク（以下、「当社」）が提供する全てのサービス（以下、「本サービス」）の利用条件を定めるものです。本サービスをご利用いただく際には、本規約に同意いただいたものとみなします。

## 1. 適用範囲

本規約は、当社が提供する全てのサービス（Bookingアプリを含むがこれに限られない）に適用されます。個別のサービスについて別途規約が定められている場合、当該規約が本規約に優先して適用されます。

1. 利用者は、本規約に同意のうえ、当社が定める方法に従い本サービスを利用するものとします。
2. 利用者が未成年者である場合、法定代理人の同意を得たうえで本サービスを利用してください。
3. 当社は、利用者に事前に通知することなく、本サービスの内容を変更、追加、または廃止することがあります。

利用者は、本サービスの利用にあたり、以下の行為を行ってはなりません。
